# 苦情対応

## 苦情の受付

### パターン
利用者家族・関係機関からの苦情・不満の申し出。

### 返信例
```
○○様

いつもお世話になっております。
この度は、ご不快な思いをおかけし、誠に申し訳ございません。

いただいた内容について、事実関係を確認のうえ、
○月○日までに改めてご報告させていただきます。

今後このようなことがないよう、改善に努めてまいります。
引き続き、何かお気づきの点がございましたらお知らせください。
```

### ポイント
- まずお詫びの気持ちを伝える
- 事実確認の報告期限を明示する
- すぐに原因を断定しない（「確認のうえ」）
- 改善の姿勢を示す

---

## 苦情への改善報告

### パターン
事実確認後、改善策を報告する。

### 返信例
```
○○様

いつもお世話になっております。
先日いただきましたご指摘について、事実確認と改善策をご報告いたします。

【確認した事実】
・（確認した内容を具体的に記載）

【原因】
・（原因を記載）

【改善策】
・（具体的な改善内容を記載）
・（実施時期を記載）

今後、同様のことが起きないよう、スタッフ一同取り組んでまいります。
この度は貴重なご意見をいただき、ありがとうございました。
```

### ポイント
- 事実→原因→改善策の順で整理
- 具体的な改善内容と時期を明示
- 最後に感謝を伝える
- 医学的判断には踏み込まない（※NGライン）
