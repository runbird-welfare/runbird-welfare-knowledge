# 他施設との連携

## 受入れ打診への回答

### パターン
他施設から「利用者を受け入れられますか？」という打診。

### 返信例
```
○○事業所 ○○様

お世話になっております。
株式会社ランバード ○○事業所の○○でございます。

ご相談いただきありがとうございます。
□□様の受入れについて、以下のとおり回答いたします。

【空き状況】
・現在の空き状況：（空きあり/空きなし/待機○名）

【受入れ条件】
・（該当する条件を記載）

まずは一度、見学にお越しいただければと思います。
ご都合のよい日時をお知らせいただけますでしょうか。

ご検討のほど、よろしくお願いいたします。
```

### ポイント
- 空き状況を明確に伝える
- 受入れ条件を率直に提示
- 見学の案内につなげる
- 他の利用者の情報には触れない（※NGライン）

## サービス担当者会議の招集

### 返信例
```
関係者各位

お世話になっております。
株式会社ランバード ○○事業所の○○でございます。

下記のとおりサービス担当者会議を開催いたしますので、
ご出席をお願い申し上げます。

【会議概要】
・日時：令和○年○月○日（○）○時〜○時
・場所：（場所 or オンライン）
・対象利用者：○○様
・議題：（主な議題）

ご出席の可否を○月○日までにご連絡いただけますと幸いです。
よろしくお願いいたします。
```

### ポイント
- 議題を事前に明示
- 出欠の回答期限を設ける
- 参加者に漏れがないか確認
