# 行政への報告

## 事故報告

### パターン
利用中の転倒・怪我等の事故が発生した場合の報告。

### 返信例
```
○○市（県）○○課 御中

平素よりお世話になっております。
株式会社ランバード ○○事業所でございます。

下記のとおり事故が発生いたしましたので、ご報告申し上げます。

【事故概要】
・発生日時：令和○年○月○日（○）○時○分頃
・発生場所：（場所）
・利用者：（氏名・年齢・障害支援区分等）
・事故の内容：（具体的な状況）
・対応内容：（実施した対応）
・現在の状態：（利用者の状態）
・再発防止策：（今後の対策）

別途、所定の事故報告書を提出いたします。
ご確認のほど、よろしくお願い申し上げます。
```

### ポイント
- 事実を正確に・時系列で記載
- 対応内容と再発防止策を必ず含める
- 報告の遅延がないよう迅速に（※自治体ごとの報告期限に注意）
- 所定の様式がある場合はそちらも提出

## 変更届

### パターン
事業所の人員変更・サービス内容変更等の届出。

### 返信例
```
○○市（県）○○課 御中

平素よりお世話になっております。
株式会社ランバード ○○事業所でございます。

下記の変更が生じましたので、届出いたします。

【変更内容】
・変更事項：（変更する項目）
・変更前：（変更前の内容）
・変更後：（変更後の内容）
・変更日：令和○年○月○日

添付書類：
・（必要な添付書類を記載）

ご確認のほど、よろしくお願い申し上げます。
```

### ポイント
- 変更前後を明確に対比
- 添付書類を漏れなく記載
- 届出期限を確認（※自治体によって異なる → 法令フォルダ参照）
