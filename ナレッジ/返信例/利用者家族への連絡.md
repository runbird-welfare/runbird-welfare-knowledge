# 利用者家族への連絡

## サービス利用条件の問い合わせ

### パターン
家族から「うちの○○はこのサービスを使えますか？」「利用条件を教えてください」等の問い合わせ。

### 返信例
```
○○様

いつもお世話になっております。
お問い合わせいただきありがとうございます。

ご質問いただいた□□サービスの利用条件について、以下のとおりご案内いたします。

【利用条件】
・（該当する条件を記載）

なお、ご利用開始にあたっては、サービス担当者会議での確認が必要となります。
詳しい手続きやスケジュールにつきましては、改めてご相談させていただければと思います。

ご不明な点がございましたら、お気軽にお問い合わせください。
```

### ポイント
- まずお礼を伝える
- 条件を箇条書きで明確に示す
- 次のステップ（手続き・会議等）を案内する
- 「お気軽に」で終える

## スケジュール調整

### パターン
通所日の変更、面談の日程調整など。

### 返信例
```
○○様

いつもお世話になっております。
ご連絡ありがとうございます。

□月□日のスケジュール変更について承知いたしました。
以下の日程で調整させていただきたいのですが、ご都合はいかがでしょうか。

・候補①：□月□日（□）□時〜
・候補②：□月□日（□）□時〜

ご確認のほど、よろしくお願いいたします。
```

### ポイント
- 変更を受け止めたことを明示
- 候補日を複数提示
- 簡潔に

## 日常の活動報告

### パターン
家族へのお子様・利用者の活動の様子の報告。

### 返信例
```
○○様

いつもお世話になっております。
本日の□□さんのご様子をお伝えいたします。

（活動内容と様子を具体的に記載）

引き続き、□□さんが安心して過ごせるようサポートしてまいります。
何かお気づきの点がございましたら、お気軽にお知らせください。
```

### ポイント
- 具体的なエピソードを入れる
- ポジティブな内容を心がける
- 医学的判断や診断に踏み込まない（※NGライン）
