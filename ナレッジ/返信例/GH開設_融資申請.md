# GH開設 — 融資・申請の返信例

## 公庫から領収書を求められた場合

### パターン
クライアントから「公庫に創業資金の領収書を求められた」と連絡があった。

### 返信例（クライアント→関係者への依頼文）
```
○○さん
○○さん

お世話になっております。
融資先候補の公庫より、創業のために使用した資金の領収書の提出を求められております。
○○さん○○万の領収書、○○事務所さん○○円の領収書は発行していただけませんでしょうか。
```

### ポイント
- コンサル費（ランバード）と行政書士費用の領収書が必要になることが多い
- クライアント自身から各担当者に依頼する形でOK
- 行政書士は即日対応してくれるケースが多い

---

## 信金・銀行へのアポイント依頼

### パターン
クライアントに地元金融機関への融資相談アポを取ってもらう必要がある。

### 返信例
```
○○様

お世話になっております。
金融機関アポイントの件です。

○○信金 ○○支店に予約電話
電話先：093-XXX-XXXX

「住宅ローンでお世話になっている○○です。
新しく会社を作りまして、創業融資のご相談をしたいのですが、
○月○日○時にお伺いすることは可能でしょうか？」

とアポイントを取得頂けると幸いです。
```

### ポイント
- 既存取引（住宅ローン等）がある金融機関が有利
- アポイントの電話スクリプトまで用意してあげるとスムーズ
- 複数の金融機関に並行してアプローチする場合もある

## 公庫オンライン申込完了の報告

### パターン
公庫のオンライン申込が完了した後、クライアントに次のアクションを伝える。

### 返信例
```
○○様

お世話になっております。
日本政策金融公庫申込完了しました。

面談の日時であったり、何かしらの返信があったら教えていただけると助かります。
よろしくお願いいたします。
```

### ポイント
- 申込完了のスクリーンショットを添付すると安心感がある
- 公庫からの連絡はクライアントのメールに届くため、転送を依頼しておく
- 登録番号は問い合わせ時に必要なので控えておく

---

## 融資面談前の資料送付

### パターン
融資面談が確定し、事前に担当者へ資料を送る必要がある。

### 返信例
```
○○様

お世話になっております。

公庫面談（○/○ ○:○○・○○様）に向けて、○○様からご指摘・追加のありました分を修正し、
資料を最新版に仕上げました。

■ 今回修正した資料

1. 創業計画書（修正版）
2. 月別収支計画書
3. 事業計画書（最新版）

面談前に、上記の書類を○○様（メールアドレス）宛にメール送付してください。
よろしくお願いいたします。
```

### ポイント
- 公庫担当者の名前・メールアドレスを明記して、クライアントが迷わないようにする
- 資料は複数回修正が入ることが多い → 「最新版」「修正版」を明記
- 部屋数変更等で事業計画が変わることがある

## 指定申請の書類案内

### パターン
指定申請に向けて必要書類をクライアントに案内する。

### 返信例
```
○○様

お世話になっております。
指定申請に向けて、下記の書類を準備・ご記入をお願いいたします。

・指定申請用情報シート（共同生活援助）
・管理者の経歴書
・サビ管の経歴書・研修証明書
・物件の設備写真
・間仕切り図面

チェックリストも添付しますので、ご確認ください。
```

### ポイント
- 自治体ごとにチェックリストが異なるため、該当自治体版を使用する
- 情報シートは複数回提出（修正）になることがある
- 設備写真（エアコン含む）は後出し対応が可能な場合もある
