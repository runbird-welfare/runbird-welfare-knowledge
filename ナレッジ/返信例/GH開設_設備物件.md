# GH開設 — 設備・物件の返信例

## エアコン設置の期限について

### パターン
クライアントから「エアコンは開設日までに全部屋設置しないといけない？」と聞かれた。

### 返信例
```
○○様

お世話になっております。
エアコン必須設備ではないのですが、前回○○市の指定申請の際に
市としては必要と言われていますので設置は必要です。

ただ、設置遅れていて写真間に合わないなどであれば
お写真を後出しで対応することも可能かとは思います！

今聞いてしまうと、間に合うようにしてくださいとなりそうなので
提出する時に説明で良さそうです。
```

### ポイント
- 法令上は必須設備ではないが、**自治体の運用で必要**とされるケースがある
- 写真の後出し対応は可能な場合がある
- **事前に自治体に聞くと厳格な回答になる**ため、提出時に説明する戦略も有効
- 自治体ごとに対応が異なるので、過去の申請実績を参照する

## 消防設備・内装工事のスケジュール

### パターン
クライアントから工事のスケジュール感を聞かれた。

### 返信例
```
○○様

お世話になっております。
目安として、消防設備工事が約1.5日、内装工事が約2日です。
開設の2〜3週間前に消防設備、その翌週に内装という流れが一般的です。

見積書が取れましたら共有をお願いいたします。
```

### ポイント
- 消防設備の見積書は早めに取得しておく
- 工事スケジュールは物件の状態により変動する
- 見積書は申請・融資資料としても使うため、早めの共有を依頼

## 物件の計測・撮影依頼

### パターン
物件候補が見つかり、計測・撮影が必要。

### 返信例
```
○○様

お世話になっております。
物件の計測・撮影依頼書をお送りいたします。
こちらの内容に沿って、計測と撮影をお願いいたします。

完了しましたら、写真と合わせてご共有ください。
```

### ポイント
- OWLの計測・撮影依頼書フォーマットを使用する
- 物件が変わる可能性もあるため、複数候補がある場合はそれぞれ依頼
- 図面は行政への事前相談（図面相談）に必要
