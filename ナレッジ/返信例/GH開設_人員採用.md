# GH開設 — 人員採用の返信例

## 管理者の採用が決まった場合

### パターン
クライアントから「管理者が採用できた。必要書類は？」と聞かれた。

### 返信例
```
○○様

お世話になっております。
管理者さんも決まったのですね！

管理者さんとしては申請時経歴書を提出します。
履歴書と経歴書があればこちらで作成可能ですので、その2点頂きたいです。

その他、資格なにかあるのであれば内容により加算の算定もありえるので
資格証もお願いいたします。
```

### ポイント
- 履歴書の顔写真は**不要**
- 資格があれば**加算の算定**につながる可能性を伝える
- 「こちらで作成可能」とフォローすることで負担を軽減する

## サビ管の採用が決まった場合

### パターン
クライアントから「サビ管が採用できた。必要書類は？」と聞かれた。

### 返信例
```
○○様

お世話になっております。
採用決まって良かったです。

下記書類お願いいたします！

・サービス管理責任者基礎研修 受講証明書
・相談支援従事者初任者研修 受講証明書
・サービス管理責任者実践研修 受講証明書
・資格証（お持ちのものあれば）
・実務経験証明書（過去の職場に依頼して貰います）
・経歴書（作成済みのものあれば、なければ履歴書職歴書）
```

### ポイント
- 更新されている方の場合は「サービス管理責任者**更新研修**」の証明書も追加依頼する
- 実務経験証明書は**過去の職場に依頼**する必要がある → 時間がかかるので早めに案内
- 書類は箇条書きで明確に列挙する

## サビ管が他事業所と兼務する場合

### パターン
クライアントから「サビ管が別の事業所でも勤務しているが、何人まで見られるか？」と聞かれた。

### 返信例（別の自治体で勤務している場合）
```
○○様

お世話になっております。
○○市なのですね。
それであれば、指定権者も異なりますのでそこまで何人見ることが出来るのかは気にされずで
実務的にどうかで良さそうです！
```

### ポイント
- **指定権者が異なる自治体**で勤務 → 人数制限は気にせず実務判断でOK
- **同一自治体**の場合 → 常勤であれば物理的に動ける時間内で可能な人数が上限
- 就労のサビ管は通常**常勤**

---

## 管理者の兼務について

### パターン
社長（オーナー）が管理者も兼務したいと相談された。

### 返信例
```
○○様

お世話になっております。
夜勤に入った時間含め、4週間で160時間クリア出来れば
管理者兼務可能です。
```

### ポイント
- 4週間で160時間 = 常勤相当の勤務実績が必要
- 現実的にハードルが高い場合は外部募集に切り替える判断も必要
- 処遇改善加算を含めた募集内容にすることが応募獲得のコツ
