# SV定期対応 — 返信テンプレート

## 定期面談後の議事メモ共有

### パターン
月1回のSV面談後、内容を共有する。

### 返信例
```
○○様

お世話になっております。
先日のお打合せもありがとうございました。
内容確認で記載させていただきます。
相違等ありましたら教えてください。

①○○について
  ・（具体的な内容）

②△△について
  ・（具体的な内容）

③次回
  ・○月○日（○）○時ご予約入れさせて頂きました。

引き続きよろしくお願いいたします。
```

### ポイント
- 議事録は**番号付き**で整理する
- 「相違等ありましたら教えてください」で認識齟齬を防ぐ
- **次回の日程**を必ず明記する
- 当日中〜翌日中には送信する

---

## 説明会の案内

### パターン
OWL定期説明会・緊急説明会の案内を送る。

### 返信例
```
○○様

【OWL福祉事業部　○○説明会開催のご案内】

いつもお世話になっております。
（説明会の趣旨を簡潔に記載）

日時　○○年○月○日(○) ○○時～
会場：オンラインZoom

ご参加いただける場合は下記のGoogleフォームにてご返信ください。
（GoogleフォームURL）

回答期限：○○年○月○日（○）23:59まで
※今回の資料に関しましては、Zoomに参加された方のみとさせていただきます。
```

### ポイント
- Googleフォームで出欠を取る
- 資料はZoom参加者のみに配布
- 回答期限を明記する

## 説明会ZoomURL送付

### パターン
説明会前日〜当日にZoomのURLを送る。

### 返信例
```
○○様

【要確認：ZoomURL送付】
○○説明会

いつもお世話になっております。
来たる○月○日(○) ○○：○○～の説明会に利用するZoomURLを
お送りしております。
お時間になりましたら、下記のリンクよりお入りくださいませ。

《入室時の注意事項》
・待機室に一旦入室していただきます。（開始15分前頃から入室可能予定）
・ご参加時のZoom名で出欠席をとりますので、企業名又はGH名にしていただき
  ますようお願いいたします。
・可能であれば、カメラはオンにしていただけると幸いでございます。
・参加される方が多いため、説明時の皆様のマイクはこちらでミュートにさせていただきます。

（ZoomURL）
ミーティングID: XXX XXXX XXXX
パスコード: XXXXXX

※資料の送付に関しましては、○日以降となりますことご了承ください。
```

### ポイント
- 入室注意事項を毎回明記する（特にZoom名を企業名にする点）
- 待機室の入室可能時間を記載
- 資料送付の時期を予告する

## 説明会後のお礼・資料送付

### パターン
説明会終了後、参加者にお礼と資料を送る。

### 返信例
```
○○様

【OWL福祉事業部　○○説明会ご参加のお礼】

いつもお世話になっております。
先日は、○○説明会にご参加いただきありがとうございました！

（要点のまとめを箇条書きで記載）

次回の説明会は、定期説明会になります。
○○年○月○日(○) ○○時～ を予定しております。
是非ご参加の程よろしくお願いいたします。
```

### ポイント
- 説明会の要点をまとめて記載する（参加者の復習用）
- 次回日程を予告する
- 資料PDFを添付する
