## 使い方（Q&Aボット）

「○○加算について教えて」と聞かれた場合：

1. **加算率・要件の正確な数値** → `_R6最新_加算減算一覧.md`（必要なら令和8年度改定は `公式資料/月次説明会/2026-04_*.md`）
2. **顧客向けの説明文・トーン** → `_文言集_前田改訂中_20251006.md`
3. **過去の月次説明会で扱われたか** → `../公式資料/月次説明会/README.md` のトピック逆引き
4. **包括的な背景** → `../公式資料/SV研修資料.md`
5. **事業所固有の対応経緯** → カテゴリ別ファイル（個別支援計画.md 等）
6. **どれにも無い場合** → 厚労省サイト（`site:mhlw.go.jp`）
7. **一般論や民間サイトでの推測は絶対禁止**
