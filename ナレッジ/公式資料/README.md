| [SV初回面談資料.md](SV初回面談資料.md) | 新規パートナーとの初回面談用資料。契約後の流れ、物件、人材、請求の概要 | SV初回面談資料 ※SV使用.pdf 101ページ |

## 使い方（Q&Aボット）

- **最新の法改正・報酬改定** → `月次説明会/` の最新回から順に参照（特に2026-04令和8年度改定）
- **直近のトピック・実務上の注意点** → `月次説明会/README.md` でトピック逆引き
- **制度・運営の包括的な質問** → `SV研修資料.md`
- **契約後すぐの基本的な質問** → `SV初回面談資料.md`
- **「いつまでに何をするか」の質問** → `OWL開設フロー_SV用雛型.md`
- **加算の説明** → 別途 `../加算/README.md` の優先順位に従う

## 更新

これらは公式資料の **静的なスナップショット** です。原本（PDF/Excel）が更新された場合は、再変換が必要です。原本は `~/Downloads/` または共有フォルダにあります。
