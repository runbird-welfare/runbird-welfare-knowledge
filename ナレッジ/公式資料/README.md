## 使い方（Q&Aボット）

- **制度・運営の質問** → まず `SV研修資料.md` を参照
- **契約後すぐの基本的な質問** → `SV初回面談資料.md` を参照
- **「いつまでに何をするか」の質問** → `OWL開設フロー_SV用雛型.md` を参照
- **加算の説明** → 別途 `../加算/_文言集_前田改訂中_20251006.md` を最優先で参照

## 更新
