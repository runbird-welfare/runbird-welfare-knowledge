# OWL月次説明会資料インデックス

OWL福祉事業（株式会社ランバード）が**毎月開催する説明会の配布資料**です。
法改正・実務上の注意点・運営トピックなど、その時々に重要な内容を扱います。

> **使い方（Q&Aボット）**
> - 「最新の○○について」と聞かれたら、**新しい月の資料から順に** 該当トピックを探す
> - 法改正や報酬改定の質問は、改定後の説明会資料が最優先
> - 古い資料は当時のスナップショット — 現行制度と異なる可能性があるので注意

## 更新方法

OWLから新しい月次説明会資料（PDF）が届いたら：

1. PDFを `~/Downloads/` に保存
2. `pdftotext -layout ファイル名.pdf` でテキスト化
3. 開催月・テーマ・トピックを確認して `公式資料/月次説明会/{YYYY-MM}_{テーマ}.md` として保存
4. このREADMEに行を追加（新しい順を維持）
